# 作者与维护

[English](../author.md) | **中文**

## 维护契约

Skill Base 由**兼职独立开发者**作为业余项目维护。

- **欢迎 Issue 和 Pull Request。**
- 回复与 review 集中在**工作日晚间和周末**，没有工作日 SLA。
- 大功能可能等维护者有空再做；小修复、范围清晰的 PR 更快。

这是刻意的：可持续节奏，比假装 7×24 on-call 更诚实。

## 参与贡献

基于 [MIT](../LICENSE) 开源。大改动不确定方向时，先开 Issue。

代码规范见仓库 [AGENTS.md](../../AGENTS.md)。
