## 常见问题解答（FAQ 文案）

**Q1：Skill Base 需要服务器吗？**
A：不需要。它可以在本地运行，也可以部署在你的任何服务器上。一行命令就能启动，不需要复杂的配置。

**Q2：支持哪些 AI IDE？**
A：目前支持 Cursor、Claude Code、Qoder、Windsurf、OpenCode 等主流工具。我们持续扩展支持范围，欢迎提出你的建议。

**Q3：数据安全吗？**
A：完全安全。Skill Base 私有化部署，数据完全存储在你自己的环境中，不会上传到任何第三方平台。

**Q4：需要付费吗？**
A：Skill Base 是开源项目，完全免费。你可以自由部署、修改和分发。

**Q5：技能可以分享给外部团队吗？**
A：可以。你可以把技能发布到公共市场，或者打包分享给其他团队。每个版本都有完整的记录和授权信息。
