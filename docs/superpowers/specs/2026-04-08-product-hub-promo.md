# Skill Base Product Hub 推广方案

## 产品概述

### 产品定位
Skill Base 是一个轻量级的团队 AI 技能分发平台，专注于让团队编码规范、业务模板、工程规范等经验，变成每个人都能安装和更新的 AI 能力。

### 核心价值
- 把团队经验变成可安装、可管理的 AI 技能
- 跨 IDE 统一分发，一个技能库支持 Cursor、Claude Code、Qoder 等多个工具
- 私有化部署，团队数据完全由自己掌控
- 版本化管理，历史版本随时可回滚

---

## 目标用户

### 主要用户
1. **技术团队负责人** - 需要统一团队编码规范和最佳实践
2. **高增长初创团队** - 快速扩张，需要标准化 AI 辅助能力
3. **独立开发者团队** - 需要管理多个项目的技能资产

### 使用场景
- 统一编码规范和代码风格
- 共享内部 API 文档和业务规则
- 管理测试用例模板和 QA 规范
- 产品需求文档（PRD）模板复用
- 前后端协作标准化流程

---

## 产品亮点

### 1. 一键部署，开箱即用
```bash
npx skill-base -d ./skill-data -p 8000
```
无需复杂配置，5 分钟即可启动服务，SQLite 自动初始化。

### 2. 跨 IDE 支持
支持 Cursor、Claude Code、Qoder、Windsurf、OpenCode 等主流 AI IDE，一套技能库服务全家。

### 3. 双端交互
- **CLI**：适合高频使用，支持搜索、安装、更新、发布
- **Web**：适合浏览协作，非研发也能轻松使用

### 4. 私有化安全
数据完全存储在你的基础设施中，内部 API、业务规则等敏感信息不会暴露到公共平台。

### 5. 版本化管理
每次发布都有版本记录，可以查看历史版本和变更日志，支持回滚到任意历史版本。
